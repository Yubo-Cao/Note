# 数据结构和算法概述

## 关系

- 数据结构是研究组织数据方式的学科， data structure。
  - 有了编程语言，就有了数据结构。
  - 学好数据结构可以编写出更漂亮，更有效的代码。
  - 学会数据结构并不意味着好代码。
- 算法是使用数据结构的学科，algorithm。学会数据结构是学会算法的必要条件。
- 程序 = 数据结构 + 算法

## 实际编程问题

- 链表实现字符串，每个节点是字符。
- 五子棋程序，判断游戏的输赢。
  - 二维数组（稀疏数组可以实现压缩）
  - 接上局
- 约瑟夫问题，丢手帕问题（环形列表）
- 公交车，查找路径（弗洛伊德），画图，修路问题（普利姆），汉诺塔（分治算法），八皇后（回溯）等。
